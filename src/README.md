# Krita Format Tests

## Eklenen Test Resimleri

`io-format-tests` klasörüne test resimleri eklendi. Bu resimler daha sonra test butonları ile yüklenecek.

## Planlanan Özellikler

- [ ] Test resimlerini butonlarla yükleyecek bir arayüz oluşturulacak
- [ ] Yüklenen her test resmi ayrı bir test senaryosu olacak
- [ ] Her test için sonuç raporlaması yapılacak
